## Multi-Agent核心逻辑

路由流程：
1. 用户提问 → 团队路由Agent
2. 路由Agent分析问题类型
3. 分配给最合适的虚拟人
4. 虚拟人基于自己的知识库回答
5. 返回结果（包含回答的虚拟人信息）

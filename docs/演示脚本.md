# AvatarX 产品演示脚本

## 演示目标
展示AvatarX的AI对话式引导和Multi-Agent协作能力

## 演示时长
5-8分钟

---

## 第一部分：AI智能引导（2分钟）

### 场景：快速创建客服虚拟人

**步骤1：打开首页**
- 访问 http://localhost:5173
- 展示简洁的AI对话界面
- 强调：无需学习复杂功能，像聊天一样创建虚拟人

**步骤2：第1轮对话**
```
AI: 你好！我是AvatarX智能助手。我会通过几个问题来了解你的需求，
    帮你创建最合适的虚拟人。

    首先，你想让虚拟人帮你做什么？比如客户服务、教育培训、业务咨询等。

你: 我想要一个电商客服虚拟人
```

**步骤3：第2轮对话**
```
AI: [根据回答动态生成问题，询问具体场景]

你: 主要处理售前咨询、订单查询和退换货问题
```

**步骤4：第3轮对话**
```
AI: [询问性格特点和能力要求]

你: 要专业、耐心、能快速理解客户需求并给出解决方案
```

**步骤5：自动创建**
- AI总结需求
- 显示"正在为你创建..."
- 自动跳转到对话页面

**亮点说明：**
- ✨ 零学习成本，3轮对话完成创建
- 🤖 AI理解自然语言，无需填写复杂表单
- ⚡ 自动生成配置，立即可用

---

## 第二部分：知识库训练（1分钟）

### 场景：上传产品文档

**步骤1：准备文档**
- 展示一个产品FAQ文档（TXT或PDF）
- 说明：虚拟人将学习这些知识

**步骤2：上传知识**
- 点击"知识训练"
- 选择刚创建的虚拟人
- 上传文档
- 展示上传成功提示

**步骤3：测试知识**
- 点击"立即对话"
- 提问文档中的内容
- 展示虚拟人基于知识库的准确回答

**亮点说明：**
- 📚 支持多种文档格式
- 🔍 自动向量化，语义检索
- 💡 RAG技术，准确回答专业问题

---

## 第三部分：Multi-Agent团队协作（3分钟）

### 场景：创建客服团队

**步骤1：创建多个虚拟人**

快速创建3个虚拟人：

1. **售前顾问**
   - 名称：小智售前
   - 人设：熟悉产品功能，善于推荐，热情主动

2. **技术支持**
   - 名称：小智技术
   - 人设：精通技术问题，逻辑清晰，解决问题能力强

3. **售后专员**
   - 名称：小智售后
   - 人设：处理退换货，安抚客户情绪，耐心细致

**步骤2：组建团队**
- 进入"团队"页面
- 创建团队："电商客服团队"
- 选择3个成员
- 系统自动设置路由规则

**步骤3：团队对话测试**

测试不同类型的问题：

**测试1：售前咨询**
```
你: 这款手机有什么特色功能？
系统: [路由到售前顾问]
售前: 这款手机有以下特色功能...
```

**测试2：技术问题**
```
你: 手机连不上WiFi怎么办？
系统: [路由到技术支持]
技术: 请按以下步骤排查...
```

**测试3：售后问题**
```
你: 我想退货，怎么操作？
系统: [路由到售后专员]
售后: 很抱歉给您带来不便，退货流程如下...
```

**亮点说明：**
- 👥 多虚拟人协作，专业分工
- 🎯 智能路由，自动分配问题
- 🔄 无缝切换，提升服务效率

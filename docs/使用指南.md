# AvatarX 产品使用指南

## 🎯 核心特色：AI对话式引导

AvatarX采用创新的AI对话引导方式，让用户通过自然对话就能创建专属虚拟人。

## 🚀 快速开始

### 方式一：AI智能引导（推荐）

1. **访问首页**
   - 打开 http://localhost:5173
   - AI助手会主动问候并开始引导

2. **3轮对话定制虚拟人**
   - **第1轮**：告诉AI你想要什么类型的虚拟人
     - 例如："我想要一个客服虚拟人"

   - **第2轮**：描述应用场景和专业领域
     - 例如："用于电商行业，处理售前咨询和售后问题"

   - **第3轮**：说明期望的性格和能力
     - 例如："专业、耐心、能快速解决问题"

3. **自动创建并开始使用**
   - AI分析你的需求
   - 自动生成虚拟人配置
   - 直接跳转到对话页面

### 方式二：传统工作台

点击"跳过引导"进入工作台，使用完整功能：
- 手动创建虚拟人
- 上传知识库
- 创建团队
- 查看数据分析

## 💡 使用技巧

### AI引导的优势
- **零学习成本**：像聊天一样创建虚拟人
- **智能理解**：AI理解你的真实需求
- **快速上手**：3轮对话即可开始使用

### 最佳实践
1. **描述要具体**：告诉AI具体的应用场景
2. **说明期望**：清楚表达对虚拟人的期望
3. **可以追问**：AI会根据你的回答调整问题

### 示例对话

**场景1：客服虚拟人**
```
AI: 你想让虚拟人帮你做什么？
你: 我需要一个客服助手

AI: 请问用于哪个行业或领域？
你: 电商行业，主要处理订单查询和退换货问题

AI: 希望虚拟人有什么特点？
你: 要专业、有耐心，能够理解客户情绪，快速解决问题
```

**场景2：教育虚拟人**
```
AI: 你想让虚拟人帮你做什么？
你: 我想要一个编程教学助手

AI: 请问主要教授哪方面的内容？
你: Python编程，面向初学者，从基础语法到实战项目

AI: 希望虚拟人有什么特点？
你: 善于用简单的例子解释复杂概念，有耐心，鼓励学生
```
